# connect4
Connect four in ruby coded for CS 3360 at UTEP. 
Test cases are provided for the different features inside
the code. There is a pop feature where the first row's piece
is removed on the specified column. The rest of the column shifts down.
The moves for the AI are prioritized in the order of the methods called
inside of robot move. To play against the row insert which column you 
would like to insert to starting at 1. Once the player or robot wins the game
will end.
